Privacy Policy

Application: Optical File Transfer

Last updated: 2026-04-21

Optical File Transfer is designed to transfer files locally using streams of QR frames. The application does not collect personal data and does not require network access for the core file transfer workflow.

Data collection

This application does not collect, sell, or share personal data with the developer. It does not send file contents or personal information to the developer's servers as part of normal use.

Permissions used by the app

The app may request the following permissions only for features directly used by the user:

  • Camera - to capture QR frame streams during optical file transfer.
  • File or storage access - to open, import, receive, and save transferred files.
  • Modify system settings - only to temporarily adjust screen brightness during transfer and then restore it.
  • Foreground service (camera) - to keep an active camera-based receive session alive while it is visibly ongoing for the user.

Advertising and analytics

Optical File Transfer does not use the Android advertising ID for advertising or analytics. The app is intended to work offline and is not designed around ad tracking.

Files and local processing

Files selected or received by the user are processed locally on the device in order to perform file transfer. The app uses local storage only to read, reconstruct, and save files involved in the transfer.

Third-party apps and links

The app can open files or links using external applications chosen by the user. When that happens, those external applications are governed by their own privacy policies.

Children's privacy

The app is not intended to collect personal data from children or from any other users.

Changes to this policy

This privacy policy may be updated if app behavior changes. The latest version will be published at this URL.

If you have questions about this privacy policy, please use the contact details provided in the app listing or developer profile.
