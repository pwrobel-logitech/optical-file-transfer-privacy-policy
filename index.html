简体中文

Optical File Transfer 是一款通过 QR 图像帧流在本地传输文件的应用。 本应用设计为离线工作，核心文件传输功能不需要网络访问，也不会与开发者服务器通信。

数据收集

本应用不会收集、出售或共享用户的个人数据。正常使用过程中，本应用不会把文件内容、 个人信息或传输内容发送给开发者服务器。

应用使用的权限

本应用仅在用户主动使用相关功能时请求和使用以下权限：

  • 相机权限 - 仅在下载/接收模式中用于扫描 QR 图像帧流，并根据这些帧重建接收到的文件。
  • 文件或存储访问 - 仅用于选择要发送的文件、读取用户选择的文件，以及保存接收到的文件。
  • 前台相机使用 - 下载/接收模式仅在本应用位于前台并可见时使用相机。 当应用暂停或移到后台时，摄像头会被释放，因此 Android 的摄像头隐私指示器应当消失。 本应用不会运行相机前台服务，也不会保留相机常驻通知。

上传/发送模式中的屏幕使用

在上传/发送模式中，本应用必须在前台保持可见，因为它会在屏幕上显示动态 QR 图像帧流，供另一台设备的相机扫描。 长时间传输时，屏幕可能保持常亮，并且本应用窗口可能临时使用较高亮度，以便接收端相机更容易读取 QR 码。 这是当前应用窗口的亮度设置；本应用不会请求或使用“修改系统设置”权限。

广告和分析

Optical File Transfer 不使用 Android 广告 ID 进行广告或分析。 本应用不是基于广告跟踪设计的应用。

文件和本地处理

用户选择或接收的文件仅在设备本地处理，用于执行文件传输、重建文件和保存文件。 本应用不会为其他目的扫描、上传或共享这些文件。

第三方应用和外部链接

本应用可以根据用户选择，请求其他应用打开文件或链接。 在这种情况下，只会向外部应用传递完成该操作所需的文件/链接信息和 MIME 类型。 外部应用或外部网站受其自身隐私政策约束。

儿童隐私

本应用不会有意收集儿童或任何其他用户的个人数据。

政策变更

如果应用行为发生变化，本隐私政策可能会更新。最新版本会发布在此页面。

如果你对本隐私政策有疑问，请通过应用商店页面或开发者资料中提供的联系方式联系开发者。

English

Optical File Transfer is designed to transfer files locally using streams of QR frames. The application is designed to work offline. The core file transfer workflow does not require network access and does not communicate with developer servers.

Data collection

This application does not collect, sell, or share personal data. During normal use, it does not send file contents, personal information, or transfer contents to developer servers.

Permissions used by the app

The app may request and use the following permissions only for features directly started by the user:

  • Camera - used only in download/receive mode to scan QR frame streams and reconstruct the received file.
  • File or storage access - used only to select files for sending, read files selected by the user, and save received files.
  • Foreground-only camera use - download/receive mode uses the camera only while this app is visible in the foreground. The camera is released when the app is paused or moved to the background, so Android's camera privacy indicator should disappear. The app does not run a camera foreground service or keep a persistent camera notification.

Screen use in upload/send mode

In upload/send mode, the app must remain visible in the foreground because it displays the selected file as an animated QR frame stream on the screen so another device can scan it with its camera. During long transfers, the screen may be kept on and this app window may temporarily use higher brightness to make QR codes easier for the receiving camera to read. This is a brightness setting for the current app window; the app does not request or use the Modify system settings permission.

Advertising and analytics

Optical File Transfer does not use the Android advertising ID for advertising or analytics. The app is not designed around ad tracking.

Files and local processing

Files selected or received by the user are processed locally on the device in order to perform file transfer, reconstruct files, and save files. The app does not scan, upload, or share those files for any other purpose.

Third-party apps and external links

The app can ask other applications to open files or links selected by the user. When this happens, only the file/link information and MIME type required for that action are passed to the external app. External apps and external websites are governed by their own privacy policies.

Children's privacy

The app is not intended to collect personal data from children or from any other users.

Changes to this policy

This privacy policy may be updated if app behavior changes. The latest version will be published at this page.

If you have questions about this privacy policy, please use the contact details provided in the app listing or developer profile.

在上傳/傳送模式中，本應用必須在前景保持可見，因為它會在螢幕上把所選檔案顯示為動態 QR 流。 長時間傳輸時，螢幕可能保持常亮，並且本應用視窗可能暫時使用最高亮度，以便接收端相機更容易讀取 QR 碼。 這只是目前應用視窗的亮度設定；本應用不會請求或使用「修改系統設定」權限。

本應用也可以請求其他應用開啟檔案或連結。在這種情況下，只會把使用者選擇操作所需的檔案/連結資訊和 MIME 類型傳遞給外部應用。

Ponieważ transmisje optyczne mogą trwać długo, tryb pobierania/odbioru oraz tryb wysyłania/nadawania są trybami transmisji pierwszoplanowej uruchamianymi przez użytkownika. W trybie pobierania/odbioru dostęp do kamery służy wyłącznie do skanowania klatek QR i odtworzenia odebranego pliku, gdy aplikacja jest widoczna na pierwszym planie. Kamera jest zwalniana, gdy aplikacja zostanie wstrzymana albo przeniesiona do tła, więc wskaźnik prywatności kamery Androida powinien zniknąć. Aplikacja nie uruchamia pierwszoplanowej usługi kamery ani nie utrzymuje stałego powiadomienia kamery.

Dostęp do plików/pamięci służy wyłącznie do wybierania plików do wysłania, odczytywania plików wybranych przez użytkownika oraz zapisywania plików odebranych na urządzeniu. Aplikacja nie skanuje, nie wysyła i nie udostępnia plików w żadnym innym celu.

W trybie wysyłania/nadawania aplikacja musi pozostać widoczna na pierwszym planie, ponieważ wyświetla wybrany plik na ekranie jako animowany strumień QR. Podczas długich transmisji ekran może pozostawać włączony, a okno tej aplikacji może tymczasowo używać maksymalnej jasności, aby kamera odbiornika łatwiej odczytywała kody QR. Jest to ustawienie jasności tylko dla okna aplikacji; aplikacja nie prosi o uprawnienie do modyfikowania ustawień systemowych i go nie używa.

Ta aplikacja zawiera linki do zewnętrznych stron, takich jak desktopowa aplikacja webowa lub strony pomocy. Jeśli otworzysz link, dana strona albo przeglądarka ma własną politykę prywatności i nie jest obsługiwana przez dewelopera.

Ta aplikacja może również prosić inne aplikacje o otwarcie pliku lub linku. W takim przypadku do zewnętrznej aplikacji wybranej przez użytkownika przekazywane są tylko wymagane informacje o pliku/linku oraz typ MIME.

Português

Aplicativo: Optical File Transfer.

Desenvolvedor: Pawel Wrobel.

Optical File Transfer não coleta, vende nem transmite dados pessoais. O aplicativo foi projetado para funcionar offline. Ele não requer acesso à rede e não se comunica com nenhum servidor.

Como as transferências ópticas podem demorar bastante, o modo de download/recebimento e o modo de upload/envio são modos de transferência em primeiro plano iniciados pelo usuário. No modo de download/recebimento, o acesso à câmera é usado apenas para escanear quadros QR e reconstruir o arquivo recebido enquanto o aplicativo está visível em primeiro plano. A câmera é liberada quando o aplicativo é pausado ou movido para segundo plano, portanto o indicador de privacidade da câmera do Android deve desaparecer. O aplicativo não executa um serviço em primeiro plano da câmera nem mantém uma notificação persistente da câmera.

O acesso a arquivos/armazenamento é usado apenas para selecionar arquivos para envio, ler os arquivos selecionados pelo usuário e salvar os arquivos recebidos no dispositivo. Os arquivos não são escaneados, enviados ou compartilhados por este aplicativo para qualquer outro fim.

No modo de upload/envio, o aplicativo deve permanecer visível em primeiro plano, porque exibe o arquivo selecionado na tela como um fluxo QR animado. Durante transferências longas, a tela pode permanecer ligada e a janela deste aplicativo pode usar temporariamente o brilho máximo para facilitar a leitura dos códigos QR pela câmera receptora. Isso é uma configuração de brilho por janela; o aplicativo não solicita nem usa permissão para modificar configurações do sistema.

Este aplicativo contém links para sites externos, como o aplicativo web para desktop/páginas de ajuda. Se você abrir um link, o site ou navegador terá sua própria política de privacidade e não é operado pelo desenvolvedor.

Este aplicativo também pode pedir que outros aplicativos abram um arquivo ou link. Nesse caso, apenas as informações necessárias do arquivo/link e o tipo MIME são passados ao aplicativo externo escolhido pelo usuário.

Русский

Приложение: Optical File Transfer.

Разработчик: Pawel Wrobel.

Optical File Transfer не собирает, не продает и не передает персональные данные. Приложение рассчитано на работу офлайн. Ему не нужен доступ к сети, и оно не связывается ни с каким сервером.

Поскольку оптическая передача может занимать много времени, режим загрузки/приема и режим отправки/передачи являются режимами передачи на переднем плане, запускаемыми пользователем. В режиме загрузки/приема доступ к камере используется только для сканирования QR-кадров и восстановления принятого файла, пока приложение видно на переднем плане. Камера освобождается, когда приложение приостанавливается или переводится в фон, поэтому индикатор конфиденциальности камеры Android должен исчезнуть. Приложение не запускает службу переднего плана камеры и не держит постоянное уведомление камеры.

Доступ к файлам/хранилищу используется только для выбора файлов для отправки, чтения выбранных пользователем файлов и сохранения принятых файлов на устройстве. Приложение не сканирует, не загружает на сервер и не передает эти файлы кому-либо для иных целей.

В режиме отправки/передачи приложение должно оставаться видимым на переднем плане, потому что оно отображает выбранный файл на экране как анимированный поток QR. При долгой передаче экран может оставаться включенным, а окно этого приложения может временно использовать максимальную яркость, чтобы принимающая камера легче считывала QR-коды. Это настройка яркости только для окна приложения; приложение не запрашивает и не использует разрешение на изменение системных настроек.

Это приложение содержит ссылки на внешние сайты, например на настольное веб-приложение или страницы справки. Если вы открываете ссылку, соответствующий сайт или браузер имеет собственную политику конфиденциальности и не управляется разработчиком.

Это приложение также может попросить другие приложения открыть файл или ссылку. В таком случае внешнему приложению, выбранному пользователем, передаются только необходимые сведения о файле/ссылке и MIME-тип.

日本語

アプリ: Optical File Transfer。

開発者: Pawel Wrobel。

Optical File Transfer は個人データを収集、販売、送信しません。このアプリはオフラインで動作するように設計されています。 ネットワークアクセスは不要で、どのサーバーとも通信しません。

光学転送には長時間かかることがあるため、ダウンロード/受信モードとアップロード/送信モードはいずれもユーザーが開始するフォアグラウンド転送モードです。 ダウンロード/受信モードでは、アプリがフォアグラウンドで表示されている間だけ、カメラアクセスは QR フレームをスキャンして受信ファイルを復元するために使用されます。 アプリが一時停止されたりバックグラウンドに移動したりするとカメラは解放されるため、Android のカメラプライバシーインジケーターは消えるはずです。このアプリはカメラ用フォアグラウンドサービスを実行せず、カメラの常時通知も維持しません。

ファイル/ストレージアクセスは、送信するファイルの選択、ユーザーが選択したファイルの読み取り、受信したファイルの端末への保存にのみ使用されます。 このアプリがその他の目的でファイルをスキャン、アップロード、共有することはありません。

アップロード/送信モードでは、選択したファイルをアニメーション QR ストリームとして画面に表示する必要があるため、アプリはフォアグラウンドで表示されたままである必要があります。 長時間の転送では画面がオンのままになり、このアプリのウィンドウが一時的に最大輝度を使用して、受信側カメラが QR コードを読み取りやすくすることがあります。 これはアプリウィンドウごとの輝度設定であり、アプリはシステム設定を変更する権限を要求または使用しません。

このアプリには、デスクトップ Web アプリ/ヘルプページなどの外部サイトへのリンクが含まれています。 リンクを開いた場合、その Web サイトまたはブラウザには独自のプライバシーポリシーがあり、開発者が運営しているものではありません。

このアプリは、他のアプリにファイルやリンクを開くよう依頼することもあります。 その場合、ユーザーが選択した外部アプリには、必要なファイル/リンク情報と MIME タイプのみが渡されます。
